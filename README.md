# Coin Flip Sequence Expected Value

This example demonstrates why the expected number of coin flips to achieve certain sequences, like 'HH' (Head-Head) versus 'HTH' (Head-Tail-Head), can be surprisingly different. It provides both a Monte Carlo simulation to empirically estimate these values and a theoretical calculation based on the overlap method, which is derived from Markov chain analysis.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
